## 주요기능

하나의 서비스로 운용하기 위한 다양한 기능들을 담고 있습니다:

- 일일 퀴즈(랜덤), 유형별 퀴즈 제공(4종류). <b>크롤링을 통한 다양한 퀴즈 데이터 확보</b>
- 웹소켓을 이용한 실시간 겨루기(👷‍♂️개발중👷‍♀️), 방 만들기, 친구 초대.
- <b>3종 소셜로그인</b>, 이메일 인증, 비밀번호 초기화 이메일 발송.
- 유저의 퀴즈 제안, 퀴즈 신고.
- <b>경험치, 레벨업, 연속학습일수, 칭호, 랭킹</b> 등 흥미요소.
